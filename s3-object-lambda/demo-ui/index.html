Supported Parameters

  • w - Width in pixels (1-2000)
  • h - Height in pixels (1-2000)
  • q - Quality (10-100)
  • f - Format (jpeg, png, webp, avif)
  • r - Rotation (0, 90, 180, 270)
  • fit - Fit mode (contain, cover, fill, inside, outside)
  • flip - Flip vertically (true/false)
  • flop - Flip horizontally (true/false)
  • grayscale - Convert to grayscale (true/false)
  • blur - Blur radius (0-100)
  • smart - Enable smart crop (true/false)
